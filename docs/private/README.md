This tree holds the author's own research, drafts, and internal notes
while scoping the Egocentric Data Collector project — the kind of
material that supports a proposal but shouldn't be handed to a
prospective client before there's a signed engagement.

## Contents

- `research/` — third-party vendors, platforms, papers, and datasets
  relevant to egocentric data collection
- `buy-vs-build/` — notes toward the buy-vs-build call referenced in
  the public proposal's Phase 1 scope
- `architecture/` — technical architecture drafts and decisions
- `meeting-notes/` — notes from calls and internal discussions
- `suppliers/` — hardware/manufacturing supplier notes

Each subdirectory has its own `README.md` explaining what belongs there.
As of this writing, these are empty placeholders — no findings,
evaluations, or conclusions have been written up yet.

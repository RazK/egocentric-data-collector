# Research (internal)

> **Internal only — not part of the public proposal.**

Reference material on the egocentric/first-person data collection
landscape: who else offers this, what platforms and tooling exist, and
what's published in the literature. This folder organizes references
for the author's own evaluation — it is not a market analysis or
competitive assessment, and none of it should be copied into the public
proposal.

- `vendors/` — companies offering egocentric/first-person data
  collection as a service
- `platforms/` — hardware/SDK platforms relevant to building the
  capture device (camera modules, SDKs, reference designs)
- `papers/` — academic papers and toolkits
- `datasets/` — existing public or licensable egocentric video datasets

Entries are logged as plain references (name, URL, one neutral line) as
they're found. None have been evaluated yet — no claims here about
quality, pricing, or fit for this project until that review happens.
